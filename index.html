Tentang Kami

Currency Converter adalah aplikasi berbasis web yang dikembangkan oleh seorang software engineer bernama Mathius Kormasela dan design advisor oleh Fadhly Fadhilah dalam rangka untuk menyelesaikan permasalahan pertukaran kurs dollar dengan rupiah. Aplikasi web ini di bangun juga bertujuan untuk mengembangkan skill pembuatnya itu sendiri. Aplikasi ini di bangun dalam rentan waktu sekita tiga hari dengan script-script yang membingungkan.

Aplikasi web ini dibangun dengan menggunakan HTML5, CSS3, Bulma Framework, JavaScript dan Jquery. Aplikasi ini dilengkapi dengan fitur cek kurs yang berguna untuk melihat harga rupiah dengan mata uang yang lainnya. Selain itu aplikasi ini juga menyertakan fitur untuk mengkonversi nilai rupiah ke nilai dollar maupun sebaliknya yang merupakan fitur utama pada aplikasi ini.
